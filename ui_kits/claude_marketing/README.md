# Claude Marketing — UI Kit

Hi-fi recreation of an Anthropic-style marketing landing page in the warm-canvas / coral-accent style. Provided as a reference for editorial-feeling marketing surfaces (the AI-history page, weekly-brief index page, and essay archive on the Midea site can all use this shell).

**Files**
- `index.html` — single-page demo: top nav, hero, eyebrow-led feature row, editorial split, three-up cards, footer.

**Surfaces covered**
- Sticky transparent top nav with brand mark + thin nav links + small coral CTA
- Editorial hero with a serif headline and a single short paragraph
- Section transitions on the cream canvas (no full-bleed images)
- Dark `--ink-900` footer with the spike mark
- All Lucide icons, all sentence case, no emoji

**Things this kit deliberately omits**
- Real product imagery (placeholders shown)
- Multi-page navigation
- Localization (Chinese pairing) — see CAVEATS in root README

Open the file directly in a browser.
